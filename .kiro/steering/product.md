# Product Overview

Bomizzel is a comprehensive ticketing system that provides both customer-facing and employee-facing interfaces for support ticket management.

## Core Purpose
- **Customer Portal**: Web interface for customers to sign up, submit tickets, and track requests
- **Employee Dashboard**: Internal interface for Bomizzel staff to manage tickets and queues

## Key Features
- Multi-company customer support with account associations
- Customizable ticket fields (picklist, string, integer, decimal, number)
- Flexible queue management with Kanban board and list views
- Real-time updates and dashboard metrics
- File attachments and ticket notes
- Email integration with automatic note creation
- Custom ticket statuses and workflows

## Target Users
- **Customers**: External users submitting support requests for their organizations
- **Bomizzel Employees**: Internal staff managing and resolving tickets
- **Team Leads**: Administrators configuring custom fields and workflows
- **Managers**: Supervisors monitoring queue performance and metrics

## Business Model
The system supports multi-tenant architecture where customers can associate with multiple companies, enabling flexible organizational structures and ticket routing.
